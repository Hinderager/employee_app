## Tech Stack

- Next.js 14 with App Router
- TypeScript
- Tailwind CSS
- Progressive Web App (PWA) enabled

## Development

```bash
# Install dependencies
npm install

# Run development server
npm run dev

# Build for production
npm run build

# Start production server
npm start
```

## Deployment

This app is deployed on Vercel and connected to the `main` branch of the GitHub repository.

**Live URL**: https://employee-app-six-blush.vercel.app

## PWA Installation

Users can install this app on their mobile devices:
- **iOS**: Tap the Share button in Safari and select "Add to Home Screen"
- **Android**: Tap the menu and select "Install App" or "Add to Home Screen"
